Requirements:
=============

  The python programs included in this packaged were written and
  tested using Python 2.7.6 running on Ubuntu 14.04

  The following extra packages are required:
  - numpy
  - bitarray:  https://pypi.python.org/pypi/bitarray

Running the attack program:
===========================

  To run the program, see comments at the top of 'bf_attack-col-filter.py'
  on the required command line arguments, and the program
  'run_bf_attack-col-filter.py' for an example set-up to run a series
  of experiments.
